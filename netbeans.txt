Abstract
--------
Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation 
der {netbeans-url}[NetBeans IDE 7.2] unter GNU/Linux.
 
Die {netbeans-url}[NetBeans IDE 7.2] bietet eine Unterstützung für verschiedene 
Programmiersprachen und -techniken. 
Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden. 

Der Artikel ist in {asciidoc-url}[AsciiDoc] erstellt worden.

[CAUTION]
====
Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend 
sein kann, und zum Anderen bestimmte Konfigurationen anders, oder 
auf einem anderem Weg durchgeführt werden können. +
Gerne nehme ich Verbesserungsvorschläge entgegen.

Es gilt wie immer: Verwendung der Anleitung auf eigene Gefahr.
====

Installation
------------
Vor der Installation von _NetBeans 7.2_ muß das OpenJDK oder das Oracle JDK
bereits installiert sein. 

Zusätzliche Software-Pakete  
sollten im Verzeichnis '/opt' (Optional) installiert werden. 

NetBeans IDE 7.2: Graphical Installer
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Installation im Verzeichnis '/opt/NetBeans/sid' (mit Benutzer 'sid')
----
$ sudo mkdir -p /opt/NetBeans/sid && chown sid.sid /opt/NetBeans/sid
$ sudo mkdir -p /opt/Tomcat/sid   && chown sid.sid /opt/Tomcat/sid    (Optional)
----

.OpenJDK
----
$ cd /opt/NetBeans/sid
$ sh netbeans-7.2-ml-linux.sh
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...
----

.Oracle JDK
----
$ cd /opt/NetBeans/sid
$ sh netbeans-7.2-ml-linux.sh --javahome /opt/Java/current/
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...
----

NetBeans IDE 7.2: Installation
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Installationsschritte (in Textform)

.Anpassen
- Basis IDE
- Java SE
- JavaScript-Debugger
- Funktionen bei Bedarf
- Apache Tomcat 7.0.27                  (Optional)
Installationsgröße: 307,7MB

.Lizenzvereinbarung
- Ich akzeptiere die Bedingungen des Lizenzabkommens

.JUnit-Lizenzvereinbarung
- Ich akzeptiere die Bedingungen des Lizenzvereinbarung JUnit installieren

.NetBeans-IDE 7.2 Installation
- /opt/NetBeans/sid/netbeans-7.2
- /usr/lib/jvm/java-6-openjdk-amd64   <-- OpenJDK
- /opt/Java/current                   <-- Oracle JDK

.Apache Tomcat 7.0.27 Installation
- /opt/Tomcat/sid/apache-tomcat-7.0.27  (Optional)

.Zusammenfassung
- NetBeans-Installationsverzeichnis: +
  /opt/NetBeans/sid/netbeans-7.2
- Apache Tomcat 7.0.27 Installationsverzeichnis: +
  /opt/Tomcat/sid/apache-tomcat-7.0.22
- JUnit wird beim ersten Start der NetBeans IDE heruntergeladen und installiert.
- Installationsgröße (gesamt): +
  307,7MB

.Einrichtung vollständig
Zum Ende der Installation werden Sie gebeten 'Anonyme Nutzungsdaten' freizugeben.
