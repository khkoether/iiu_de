Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation des Sun Java SDK 6 Update 17 und der NetBeans IDE 6.8 unter GNU/Linux.

Die NetBeans IDE bietet eine Unterstützung für verschiedene Programmiersprachen und -techniken. Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden.

Der Artikel ist in ASCII DOC erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können.
Gerne nehme ich Verbesserungsvorschläge entgegen.

Keine Haftung für Schäden, die durch die Anwendung der aufgeführten Kommandos entstehen.

Vorraussetzungen

Die Installation wird für Ubuntu 9.10 (Karmic Koala) beschrieben. Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Download

  • Sun Java SDK 6 Update 17
    jdk-6u16-linux-i586.bin (ca. 79 MB)

  • NetBeans 6.8
    netbeans-6.8-ml-linux.sh (ca. 246 MB) (IDE-Sprache: Deutsch)

Die Installation des Sun Java SDK wird vom Benutzer root für das System durchgeführt.
Während anschließende Installation der NetBeans IDE für den Benutzer bob ausgeführt wird.

$ ls -l
-rw-r--r-- 1 bob bob  81792634 2009-12-11 21:26 jdk-6u17-linux-i586.bin
-rw-r--r-- 1 bob bob 257131520 2009-12-11 22:11 netbeans-6.8-ml-linux.sh

Installation

Sun Java SDK 6 Update 17

Zusätzliche Software-Pakete die nicht über das Paketmanagement verwaltet wird, sollte im Verzeichnis /opt (Optional) installiert werden.

Step 1: Installation im Verzeichnis /opt/Java (mit Benutzer root)

$ cd /opt
$ mkdir Java && cd Java
$ sudo sh jdk-6u16-linux-i586.bin  <--- Lizenbedingungen bestätigen!
$ ls -l
drwxr-xr-x 10 root root 4096 2009-08-30 17:37 jdk1.6.0_16

Step 2: Java-Installation für den Mozilla Firefox einrichten

$ cd /usr/lib/firefox-addons/plugins/
$ ls
libflashplayer.so  nppdf.so
$ sudo ln -s ../../../../opt/Java/jdk1.6.0_17/jre/plugin/i386/ns7/libjavaplugin_oji.so
$ ls -l
lrwxrwxrwx 1 root root     73 2009-12-11 22:18 libjavaplugin_oji.so -> ../../../../opt/Java/jdk1.6.0_17/jre/plugin/i386/ns7/libjavaplugin_oji.so
[...]

Browser> about:plugins

NetBeans IDE 6.8

Step 1: Installation im Verzeichnis /opt/NetBeans/bob (mit Benutzer bob)

$ cd /opt
$ mkdir -p NetBeans/bob && cd NetBeans/bob
$ sh netbeans-6.8-ml-linux.sh --javahome /opt/Java/jdk1.6.0_17/

Step 5: NetBeans IDE: Ruby Platform Manager
JRuby 1.4.0 (Built-in)
RubyGems 1.3.5

Der Eintrag (Platform) Ruby 1.8.7-p174 ist die bereits von mir im Linux-System installierte C-Ruby-Version. Die PATH-Variable wird während der Installation ausgewertet und vorhandene Ruby-Installationen automatisch in der NetBeans IDE eingetragen.

Der Eintrag (Platform) Ruby 1.9.1-p376 muß über den Button Add Platform vorgenommen werden.
