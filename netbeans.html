Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation der NetBeans IDE 7.3.1 unter GNU/Linux.

Die NetBeans IDE 7.3.1 bietet eine Unterstützung für verschiedene Programmiersprachen und -techniken. Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden.

Der Artikel ist in AsciiDoc erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können.
Gerne nehme ich Verbesserungsvorschläge entgegen.

Es gilt wie immer: Verwendung der Anleitung auf eigene Gefahr.

Vorraussetzungen

Die Installation wird für Ubuntu 12.04 LTS (Precise Pangolin) beschrieben.
Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Ubuntu 12.04 LTS (Precise Pangolin) enthält das Paket netbeans_7.0.1+dfsg1-4_all.

Installation

Zusätzliche (große) Software-Pakete sollten im Verzeichnis /opt (Optional) installiert werden.

NetBeans IDE 7.3.1: Graphical Installer

Installations-Verzeichnisse anlegen
$ sudo mkdir -p /opt/NetBeans/sid && chown sid.sid /opt/NetBeans/sid   

$ sudo mkdir -p /opt/Tomcat/sid   && chown sid.sid /opt/Tomcat/sid
Das Anlegen der Verzeichnisse in /opt muß der Benutzer root durchführen.
Optional, falls das Software-Paket Tomcat installiert werden sollte.

OpenJDK

$ cd /opt/NetBeans/sid
$ sh netbeans-7.3.1-ml-linux.sh
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...

Oracle JDK

$ cd /opt/NetBeans/sid
$ sh netbeans-7.3.1-ml-linux.sh --javahome /opt/Java/current/
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...

Start und Konfiguration

NetBeans IDE 7.3.1

Step 1: netbeans ausführen …

$ /opt/NetBeans/sid/netbeans-7.3.1/bin/netbeans

NetBeans IDE 7.3.1: Plugins

Step 2: NB IDE → Tools → Plugins

 
Step 3: Plugins Check for Updates

 
Step 4: Plugins Downloaded → Add Plugins…

 
Step 5: Add Plugins updates → Alle Dateien markieren → Öffnen
Hinweis:   updates   ist das mit dem Befehl   unzip   erstellte Verzeichnis.

 
Step 6: Plugins Downloaded (24) → Install

 

NetBeans IDE 7.3.1: Neustart

Step 7: My NetBeans

NetBeans IDE 7.3.1: Ruby Platforms

Step 8: NB IDE → Tools → Ruby Platforms

 
Step 9: Ruby Platform Manager

Platforms

  • Der Eintrag Built-in JRuby 1.7.3 ist Teil der Installation des Plugins Ruby on Rails.

  • Der Eintrag JRuby 1.7.5 ist die von mir bereits zuvor installierte JRuby-Version.

  • Der Eintrag Ruby 1.8.7-p352 ist die in Ubuntu 12.04 standardmäßig installierte C-Ruby-Version.

Die PATH-Variable wird während der Installation ausgewertet und vorhandene Ruby-Installationen automatisch in der NetBeans IDE 7.3.1 eingetragen.

Im PATH war zum Zeitpunkt der Installation kein Verweis auf die von mir zusätzlich installierten Versionen Ruby 1.9.3 und Ruby 2.0.0!

 
Step 10: Ruby Platform Manager → Add Platform → Ruby 2.0.0-p247

NetBeans IDE 7.3.1: JRuby 1.7.5 Gems

Step 11: Ruby Platform Manager → Gem Manager
