Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation von JRuby 1.4.0 unter GNU/Linux.

Die Programmiersprache JRuby ist die Implementation von Ruby in Java, die wiederum die Grundlage für das WebApplication Framework Ruby on Rails bildet.

Der Artikel ist in ASCII DOC erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können.
Gerne nehme ich Verbesserungsvorschläge entgegen.

Keine Haftung für Schäden, die durch die Anwendung der aufgeführten Kommandos entstehen.

Vorraussetzungen

Die Installation wird für Ubuntu 9.10 (Karmic Koala) beschrieben. Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Installation

JRuby 1.4.0

Zusätzliche Software-Pakete die nicht über das Paketmanagement verwaltet wird,
sollte im Verzeichnis /opt (Optional) installiert werden.

Step 1: Installation im Verzeichnis /opt/JRuby (mit Benutzer bob)

$ cd /opt/JRuby/bob
$ mkdir tmp && cd tmp
$ tar xvzf jruby-1.4.0.tar.gz
$ mv jruby-1.4.0 ../jruby-1.4.0-23
$ cd .. && rm -r tmp
$ ls -l
drwxr-xr-x 8 bob bob 4096 2009-12-12 21:58 jruby-1.4.0-23

Step 2: JRuby-Pfad setzen (von außen) Erstellen Sie eine Datei jruby.path.140-23.sh (siehe [Anhang]).

Achtung: Ausführen der Datei mit dem Punkt-Operator!

$ . jruby.path.140-23.sh

Anhang

Script zum Setzen der Umgebung von JRuby v1.4.0 (ohne weitere Erläuterung)
JAVA_BINDIR=/opt/Java/jdk1.6.0_17/bin
JAVA_HOME=/opt/Java/jdk1.6.0_17
JDK_HOME=/opt/Java/jdk1.6.0_17
JRE_HOME=/opt/Java/jdk1.6.0_17

JRUBY_HOME=/opt/JRuby/bob/jruby-1.4.0-23
CATALINA_OPTS='-server -Xms512m -Xmx1024m -XX:PermSize=256m -XX:MaxPermSize=512m'
PATH=$JAVA_BINDIR:$JRUBY_HOME/bin:$PATH

export JAVA_BINDIR
export JAVA_HOME
export JDK_HOME
export JRE_HOME
export JRUBY_HOME
export CATALINA_OPTS
export PATH
