Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation der NetBeans IDE 7.2 unter GNU/Linux.

Die NetBeans IDE 7.2 bietet eine Unterstützung für verschiedene Programmiersprachen und -techniken. Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden.

Der Artikel ist in AsciiDoc erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können.
Gerne nehme ich Verbesserungsvorschläge entgegen.

Es gilt wie immer: Verwendung der Anleitung auf eigene Gefahr.

Vorraussetzungen

Die Installation wird für Ubuntu 12.04 LTS (Precise Pangolin) beschrieben.
Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Ubuntu 12.04 LTS (Precise Pangolin) enthält das Paket netbeans_7.0.1+dfsg1-4_all.

Installation

Vor der Installation von NetBeans 7.2 muß das OpenJDK oder das Oracle JDK bereits installiert sein.

Zusätzliche Software-Pakete die nicht über das Paketmanagement verwaltet wird, sollten im Verzeichnis /opt (Optional) installiert werden.

NetBeans IDE 7.2: Graphical Installer

Installation im Verzeichnis /opt/NetBeans/sid (mit Benutzer sid)

$ sudo mkdir -p /opt/NetBeans/sid && chown sid.sid /opt/NetBeans/sid
$ sudo mkdir -p /opt/Tomcat/sid   && chown sid.sid /opt/Tomcat/sid    (Optional)
OpenJDK
$ cd /opt/NetBeans/sid
$ sh netbeans-7.2-ml-linux.sh
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...
Oracle JDK
$ cd /opt/NetBeans/sid
$ sh netbeans-7.2-ml-linux.sh --javahome /opt/Java/current/
Konfiguriere Installation...
Suche verfügbare JVM auf dem System...
Extrahiere Installationsdaten...
Starte Installationassistent...

Zusammenfassung

  • NetBeans-Installationsverzeichnis:
    /opt/NetBeans/sid/netbeans-7.2

  • Apache Tomcat 7.0.27 Installationsverzeichnis:
    /opt/Tomcat/sid/apache-tomcat-7.0.22

  • JUnit wird beim ersten Start der NetBeans IDE heruntergeladen und installiert.

  • Installationsgröße (gesamt):
    307,7MB
