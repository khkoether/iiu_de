Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation des Sun Java SDK 6 und der NetBeans IDE 6.9 unter GNU/Linux.

Die NetBeans IDE bietet eine Unterstützung für verschiedene Programmiersprachen und -techniken. Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden.

Der Artikel ist in ASCII DOC erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können.
Gerne nehme ich Verbesserungsvorschläge entgegen.

Es gilt wie immer: Verwendung der Anleitung auf eigene Gefahr.

Vorraussetzungen

Die Installation wird für Ubuntu 10.04 (Lucid Lynx) beschrieben. Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Download

  • Sun Java SDK 6 Update 21
    jdk-6u18-linux-i586.bin (ca. 81 MB)

  • NetBeans 6.9
    netbeans-6.9-ml-linux.sh (ca. 250 MB) (IDE-Sprache: Deutsch)

Die Installation des Sun Java SDK wird vom Benutzer root für das System durchgeführt.
Während anschließende Installation der NetBeans IDE für den Benutzer bob ausgeführt wird.

$ ls -l
-rw-r--r-- 1 bob bob 83854743 2010-07-09 18:31 jdk-6u21-linux-i586.bin
-rw-r--r-- 1 bob bob 261880832 2010-06-20 09:52 netbeans-6.9-ml-linux.sh

Installation

Sun Java SDK 6 Update 21

Zusätzliche Software-Pakete die nicht über das Paketmanagement verwaltet wird, sollten im Verzeichnis /opt (Optional) installiert werden.

Step 1: Installation im Verzeichnis /opt/Java (mit Benutzer root)

$ sudo mkdir /opt/Java && cd /opt/Java
$ sudo sh jdk-6u21-linux-i586.bin  <--- Lizenbedingungen bestätigen!
$ ls -l
drwxr-xr-x 10 root root 4096 2010-06-15 19:08 jdk1.6.0_21

Step 2: Java-Installation für den Mozilla Firefox einrichten

$ cd /etc/alternatives
$ sudo ln -s /opt/Java/jdk1.6.0_21/jre/plugin/i386/ns7/libjavaplugin_oji.so mozilla-javaplugin.so
$ cd /usr/lib/mozilla/plugins
$ sudo ln -s /etc/alternatives/mozilla-javaplugin.so libjavaplugin.so
$ ls -l
lrwxrwxrwx 1 root root     39 2010-07-09 18:44 libjavaplugin.so -> /etc/alternatives/mozilla-javaplugin.so
[...]

Browser> about:plugins
Browser> 3D-Körper

NetBeans IDE 6.9

Step 1: Installation im Verzeichnis /opt/NetBeans/bob (mit Benutzer bob)

$ sudo mkdir -p /opt/NetBeans/bob && chown bob.bob /opt/NetBeans/bob
$ sudo mkdir -p /opt/Tomcat/bob   && chown bob.bob /opt/Tomcat/bob    (Optional)

$ cd /opt/NetBeans/bob
$ sh netbeans-6.9-ml-linux.sh --javahome /opt/Java/jdk1.6.0_21/

Installation abgeschlossen

Zum Ende der Installation werden Sie zum Einen gebeten Anonyme Nutzungsdaten freizugeben, und zum Anderen sich bei der Fa. SUN zu registrieren. Beides ist freiwillig.

$ ls -la .netbeans*

Mit der superID wird die Installation sowohl auf dem Rechner als auch für den SDN Account bei der Fa. SUN eindeutig benannt.

 
 
Step 4: NetBeans IDE starten

$ /bin/sh "/opt/NetBeans/bob/netbeans-6.9/bin/netbeans"

Anhang

Script zum Setzen der Umgebung für das Built-in JRuby 1.5.0 (ohne weitere Erläuterung)
JAVA_BINDIR=/opt/Java/jdk1.6.0_21/bin
JAVA_HOME=/opt/Java/jdk1.6.0_21
JDK_HOME=/opt/Java/jdk1.6.0_21
JRE_HOME=/opt/Java/jdk1.6.0_21

JRUBY_HOME=/opt/NetBeans/bob/netbeans-6.9/ruby/jruby-1.5.0/bin
CATALINA_OPTS='-server -Xms512m -Xmx1024m -XX:PermSize=256m -XX:MaxPermSize=512m'

PATH=$JAVA_BINDIR:$JRUBY_HOME/bin:$PATH

export JAVA_BINDIR
export JAVA_HOME
export JDK_HOME
export JRE_HOME
export JRUBY_HOME
export CATALINA_OPTS
export PATH
