Abstract

Git ist ein freies verteiltes Versionskontrollsystem (DVCS - Distributed Version Control System).

Der Artikel ist eine Schritt-für-Schritt Anleitung zur Installation des Programms Git aus dem QuellCode.

Git ist Open Source und ist freigegeben unter der GPL v2-Lizenz.

Der Artikel ist in AsciiDoc geschrieben und mit Asciidoctor 1.5.5 erstellt worden.

Zu beachten ist, dass so ein Dokument zum Einen nie abschliessend sein kann, und zum Anderen bestimmte Konfigurationen anders, oder auf einem anderem Weg durchgeführt werden können. Gerne nehme ich Verbesserungsvorschläge entgegen.

Es gilt wie immer: Verwendung der Anleitung auf eigene Gefahr.

Vorraussetzung

Die Installation wird für Ubuntu 16.04 LTS (Xenial Xerus) beschrieben.
Gleichwohl sind die einzelnen Schritte auch für andere Debian-basierte GNU/Linux Distributionen übertragbar.

Ubuntu 16.04 LTS (Xenial Xerus) enthält das Paket

git-core_2.7.4-0ubuntu1.3_all.deb.

Installation

Zur Installation von Git muß auf dem Rechner autoconf und auch weitere Pakete installiert sein.

$ sudo apt-get install ...   
               autoconf build-essential libc6 libcurl3-gnutls
               libexpat1 libexpat1-dev
               perl-modules liberror-perl libdigest-sha-perl
               libcurl4-openssl-dev libssl-dev
               tcl8.4  tcl8.4-dev
               xmlto zlib1g
Der GCC ist standardmäßig bereits vorhanden.

Ohne die Angabe eines Verzeichnisses wird /usr/local/…​ automatisch verwendet.
Eine bereits vorhandene Git-Installation wird überschrieben.

Beispiel für die Änderung des Installations-Verzeichnisses:
./configure --prefix=/opt/wo_auch_immer

Step 1: Auspacken des Source-Code

$ tar xvzf git-2.16.2.tar.gz
$ cd git-2.16.2

Step 2: Konfiguration erstellen und kompilieren

$ make configure | tee ../log.git_make-configure
GIT_VERSION = 2.16.2

Anhang

Script zum Installieren von Git 2.16.2 (ohne weitere Erläuterung).
Die Dokumentation zu Git wird mit diesem Script nicht installiert.

Datei: git-install.sh
#!/bin/bash
#
ME=make_git
NAME=git-2.16.2

ls -l git*.tar.gz
echo

echo "Git-Installation fortsetzen (j/n)"
read dummy
case $dummy in
  j|J|y|Y) echo "Installation wird fortgesetzt."
           ;;
  *) echo "Installation wird abgebrochen."
     exit 1
     ;;
esac

echo
echo "Source auspacken ..."
tar xvzf ${NAME}.tar.gz
cd ${NAME}

echo
echo "Source übersetzen ..."
make configure | tee ../log.git_make-configure
./configure | tee ../log.git_configure
make | tee ../log.git_make

echo
echo "Git installieren"
sudo make install | tee ../log.git_make-install

echo "Dokumentation fehlt!"

echo
cd /usr/local/bin
ls -l git*

echo
echo "$ME: Ende."
