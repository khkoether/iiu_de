Abstract

Dieser Artikel ist eine Schritt-für-Schritt Anleitung zur Installation des Sun Java SDK 6 Update 16 und der NetBeans IDE 6.7.1 unter GNU/Linux.

Die NetBeans IDE bietet eine Unterstützung für verschiedene Programmiersprachen und -techniken. Wobei die Java-basierten Lösungen ausgezeichnet unterstützt werden.

Der Artikel ist in ASCII DOC erstellt worden.

Der Artikel wurde nach bestem Wissen und Gewissen geschrieben und geprüft. Ich übernehme jedoch keinerlei Haftung für Schäden, die durch Ausführung der in diesem Artikel aufgeführten Kommandos entstehen oder entstanden sind.

Vorraussetzungen

Die Installation wird für Ubuntu 9.04 (Jaunty Jackolope) beschrieben. Gleichwohl sind die einzelnen Schritte auch für andere Distributionen übertragbar.

Download

  • Sun Java SDK 6 Update 16
    jdk-6u16-linux-i586.bin (ca. 78 MB)

  • NetBeans 6.7.1
    netbeans-6.7.1-ml-linux.sh (ca. 248 MB) (IDE-Sprache: Deutsch)

Die Installation des Sun Java SDK wird vom Benutzer root für das System durchgeführt.
Während anschließende Installation der NetBeans IDE für den Benutzer bob ausgeführt wird.

$ ls -l
-rw-r--r-- 1 bob bob  81790339 2009-08-30 17:20 jdk-6u16-linux-i586.bin
-rw-r--r-- 1 bob bob 259429376 2009-09-17 12:34 netbeans-6.7.1-ml-linux.sh

Installation

Sun Java SDK 6 Update 16

Zusätzliche Software-Pakete die nicht über das Paketmanagement verwaltet wird, sollte im Verzeichnis /opt (Optional) installiert werden.

Step 1: Installation im Verzeichnis /opt/Java (mit Benutzer root)

$ cd /opt
$ mkdir Java && cd Java
$ sudo sh jdk-6u16-linux-i586.bin  <--- Lizenbedingungen bestätigen!
$ ls -l
drwxr-xr-x 10 root root 4096 2009-08-30 17:37 jdk1.6.0_16

Step 2: Java-Installation für den Mozilla Firefox einrichten

$ cd /usr/lib/firefox-addons/plugins/
$ sudo ln -s ../../../../opt/Java/jdk1.6.0_16/jre/plugin/i386/ns7/libjavaplugin_oji.so
$ ls -l
lrwxrwxrwx 1 root root     73 2009-08-30 17:44 libjavaplugin_oji.so -> ../../../../opt/Java/jdk1.6.0_16/jre/plugin/i386/ns7/libjavaplugin_oji.so
[...]

Browser> about:plugins

NetBeans IDE 6.7.1

Step 1: Installation im Verzeichnis /opt/NetBeans/bob (mit Benutzer bob)

$ cd /opt
$ mkdir -p NetBeans/bob && cd NetBeans/bob
$ sh netbeans-6.7.1-ml-linux.sh --javahome /opt/Java/jdk1.6.0_16/
